# Short Answers

## 1. Difference between Google Knowledge Graph and Knowledge Panel
The Knowledge Graph is Google's internal database of entities and their relationships.  
The Knowledge Panel is the visual display of that data shown in search results.

---

## 2. How Google Determines Entity Identity
Google uses:
- Structured data (schema markup)
- Consistent naming across platforms
- Backlinks and mentions
- Social profiles
- Authoritative sources

Consistency and repetition across trusted sources strengthen entity identity.

---

## 3. When to Create Custom Post Types Instead of Pages
Custom Post Types should be used when:
- Content is structured and repeatable (e.g., services, portfolios, testimonials)
- You need custom fields and filtering
- You want scalability and better content organization

Pages are better for static content like About or Contact.

---

## 4. Recommended Plugins for Speed Optimization and Why
- LiteSpeed Cache: Advanced caching and server-level optimization
- WP Rocket: Easy setup and powerful performance features
- Autoptimize: CSS, JS, and HTML optimization
- Smush or ShortPixel: Image compression

These plugins improve load speed, reduce file size, and enhance Core Web Vitals.
